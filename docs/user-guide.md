# 剑网三副本管家 使用手册

JX3 Raid Manager 是一款专为《剑网三》玩家打造的本地化副本数据管理工具，帮助你记录副本收入、管理账号角色，并支持自动扫描游戏数据。

## 快速开始

### 第一步：下载安装

前往 [GitHub Releases](https://github.com/lesswhite810/jx3-raid-manager/releases) 页面下载最新版本：

- **安装版 (NSIS)**：自动安装到开始菜单，支持应用内自动更新（推荐）
- **便携版**：直接运行，无需安装，不支持自动更新

### 第二步：配置游戏路径

1. 点击底部导航栏的 **配置** 图标
2. 在"游戏路径设置"中，点击选择你的《剑网三》安装根目录
   - 例如：`E:\Game\SeasunGame`
   - 程序会自动补全到 `Game\JX3\bin\zhcn_hd`

### 第三步：添加账号

有两种方式添加账号：

**自动扫描（推荐）**
1. 进入 **账号管理** 页面
2. 点击右上角 **扫描账号** 按钮
3. 程序会自动读取游戏目录中的账号和角色数据

**手动添加**
1. 进入 **账号管理** 页面
2. 点击 **添加账号**
3. 输入账号名称，选择账号类型（本人/代清）
4. 点击账号卡片上的 **添加角色**，输入角色信息

## 功能介绍

### 首页概览

首页展示你的核心数据，包括：

| 统计项 | 说明 |
|--------|------|
| **总金币收入** | 本周期内的所有副本收入总和 |
| **副本记录数** | 本周期内记录的副本数量 |
| **玄晶统计** | 本周期内获得的玄晶数量 |
| **本期欧皇** | 本周期内单次收入最高的角色 |
| **本期败家子** | 本周期内单次支出最高的角色 |

**时间筛选**：支持按"本周"、"本月"、"全部"切换统计周期。

**副本收益分布图**：以饼图形式展示各副本的收入占比。

**试炼翻牌统计**：展示试炼之地的翻牌数据汇总，点击可查看详细统计。

### 账号管理

账号管理页面用于统一管理你的游戏账号和角色。

#### 功能说明

| 功能 | 操作 |
|------|------|
| **添加账号** | 点击"添加账号"按钮，输入账号信息 |
| **添加角色** | 点击账号卡片上的"添加角色"按钮 |
| **编辑账号/角色** | 点击账号/角色卡片上的编辑图标 |
| **删除账号/角色** | 点击账号/角色卡片上的删除图标 |
| **拖动排序** | 长按账号卡片并拖动，可以自定义账号顺序 |
| **筛选账号** | 按"全部 / 本人 / 代清"快速筛选 |
| **搜索** | 支持按角色名、区服、账号名搜索 |
| **扫描账号** | 点击右上角扫描按钮，自动导入游戏目录中的账号数据 |

#### 账号类型

- **本人账号**：标记为自己所有的账号
- **代清账号**：标记为帮别人清号的账号，可在备注中补充代清信息

#### 角色信息

每个角色可记录以下信息：

- 角色名称
- 服务器
- 门派
- 装备分数（可选）
- 备注（可选）

### 副本管理

副本管理是核心功能，用于记录每次副本的收入。

#### 添加副本记录

1. 进入 **副本管理** 页面
2. 点击 **添加记录** 按钮
3. 填写以下信息：
   - **角色**：从已添加的角色中选择
   - **副本**：选择副本名称（支持收藏常用副本）
   - **难度**：普通 / 英雄 / 挑战
   - **收入**：输入本次获得的工资
   - **时间**：选择副本日期和时间（默认为当前时间）
   - **备注**：可选，添加补充说明

#### 查看副本详情

点击任意副本记录卡片，可以查看：

- 该角色在此副本的所有历史记录
- 每个 BOSS 的击杀状态（CD 追踪）
- 收支明细和掉落物记录

#### BOSS CD 追踪

每个副本记录可以单独追踪 BOSS 的 CD 状态：

- **25 人副本**：每周一 7:00 刷新
- **10 人副本**：每周一、周五 7:00 刷新

卡片上会显示距离下次刷新的倒计时。

#### 收藏副本

在副本列表中，点击副本卡片上的星标可以收藏常用副本，收藏的副本会显示在列表顶部，方便快速访问。

### 百战管理

百战管理用于记录百战异闻录的收入和支出。

#### 添加百战记录

1. 进入 **百战管理** 页面
2. 点击 **添加记录** 按钮
3. 选择角色、百战名称
4. 输入收入/支出金额
5. 选择时间（默认为当前时间）

#### 查看角色百战历史

点击角色卡片上的 **历史** 图标，可以查看该角色所有百战记录的详细历史。

---

### 试炼之地

试炼之地功能用于记录试炼之地的通关数据。

#### 添加试炼记录

1. 进入 **试炼之地** 页面
2. 点击 **添加记录** 按钮
3. 选择角色
4. 选择通关层数
5. 选择翻牌位置（1-5号位）
6. 如果翻中获得装备，选择装备名称（支持搜索）
7. 选择时间（默认为当前时间）

#### 翻牌统计

试炼之地页面会展示翻牌统计数据，包括：

- 各位置的翻牌次数
- 翻中装备率
- 位置出现率

点击 **查看详情** 可以进入独立的统计详情页面，查看全部历史汇总数据。

#### 查看角色试炼历史

点击角色卡片上的 **历史** 图标，可以查看该角色所有试炼记录的详细历史。

## 配置设置

点击底部导航栏的 **配置** 图标进入设置页面。

### 游戏路径设置

设置《剑网三》的游戏安装目录，用于自动扫描账号数据。

**注意**：
- 程序会自动补全到 `Game\JX3\bin\zhcn_hd`
- 如果已配置过旧格式路径，程序会自动兼容处理

### 主题切换

支持 **深色模式** 和 **浅色模式** 切换，点击即可即时生效。

### 数据目录配置

- 查看当前数据库和日志文件的存储位置
- 支持切换自定义目录
- 支持恢复默认目录

### 更新检查

- **安装版**：支持应用内自动更新，检测到新版本后会弹出更新提示
- **便携版**：检测到新版本后会提示前往发布页手动下载
- 如 GitHub 不可用，会自动回退到 Gitee 同步源

### 版本信息

显示当前应用版本号和构建时间。

## 游戏内设置

### 录像设置

在游戏内 **设置 → 录像设置** 中可以配置录像相关选项：

| 设置项 | 说明 |
|--------|------|
| **录像模式** | 可选择开启或禁用录像功能 |
| **保存录像数量** | 最多保留的录像文件数量 |
| **保存录像天数** | 录像文件保留的天数 |
| **保存录像位置** | 录像文件的存储目录 |

> 录像文件会占用磁盘空间，建议根据需要合理设置保存数量和天数，或直接禁用录像功能以节省空间。

### 推荐录像设置

如果你不需要自动录像，建议：

1. 将 **录像模式** 设置为 **禁用**
2. 或将 **保存录像数量** 和 **保存天数** 设置为较小的值

这样可以避免录像文件占用过多磁盘空间。

## 常见问题

### Q: 为什么扫描账号没有数据？

**A**: 请确保：
1. 已正确配置游戏路径
2. 游戏目录存在且可访问
3. 游戏目录下有有效的账号数据文件

### Q: 数据存储在哪里？

**A**: 应用数据存储在用户目录下：
- **安装版**：`%APPDATA%\JX3RaidManager\`
- **便携版**：程序所在目录下的 `JX3RaidManager\` 目录

包含：
- `data.db` - 数据库文件
- `logs/` - 日志文件目录

### Q: 如何备份数据？

**A**: 只需备份上述数据目录即可，包含完整的账号信息、记录数据和配置信息。

### Q: 卸载后数据会自动删除吗？

**A**:
- **安装版**：卸载时可以选择是否删除数据库和日志文件
- **便携版**：直接删除程序目录即可（需手动删除数据目录）

### Q: 副本 CD 刷新时间是什么？

**A**:
- **25 人副本**：每周一 7:00 刷新
- **10 人副本**：每周一、周五 7:00 刷新

### Q: 支持自动更新吗？

**A**:
- **安装版**：支持应用内自动更新
- **便携版**：不支持自动更新，需手动下载新版本

### Q: 界面显示异常怎么办？

**A**: 尝试以下步骤：
1. 检查是否使用了最新版本
2. 清除浏览器缓存（如果是 Web 版）
3. 检查游戏路径配置是否正确
4. 查看日志文件排查问题
