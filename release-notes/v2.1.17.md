# 剑网三副本管家 v2.1.17

## 修复 bug

- 优化账号管理拖动动画，拖拽过程更跟手，账号重排过渡更顺滑
- 修复概览页稀有掉落详情只显示玄晶的问题，现可完整查看全部特殊掉落记录
- 补齐账号管理筛选能力，现可按“全部 / 本人 / 代清”快速筛选账号列表
- 补齐数据目录配置，现可在配置页查看当前数据目录并切换自定义目录或恢复默认目录
- 修复安装版卸载时无法按勾选删除数据库与日志的问题，并补齐中文卸载提示
- 修复安装版浏览安装目录时的路径处理，选择目录后会自动追加应用文件夹名称
- 优化安装版首次安装流程，安装完成后会自动迁移旧用户目录中的数据库与日志，并在安装阶段先完成数据库初始化与升级
- 优化页面搜索快捷键，在带搜索框的页面按 `Ctrl+F` 会直接聚焦当前搜索框，减少原生页面搜索卡顿
