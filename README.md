# JX3 Raid Manager（剑网三副本管家）

> 一款为《剑网3》团长与多号玩家打造的本地化团队管理工具，集中解决副本收入记录、账号角色管理、掉落统计、BOSS CD 追踪等日常痛点。

![JX3 Raid Manager 预览图](https://cdn.jx3box.com/upload/post/2026/2/24/29953_1557557.png)

## ✨ 核心功能

- **💰 智能工资结算** - 记录团队副本、百战、试炼之地等玩法的收入与支出，支持备注、明细查看与汇总统计
- **📊 数据概览更清晰** - 首页支持按时间范围查看收益、记录数、稀有掉落等关键数据
- **👥 账号与角色统一管理** - 支持手动添加或扫描游戏目录自动识别本地账号，支持本人/代清分类管理
- **↕️ 账号拖动排序** - 账号管理页支持直接拖动账号栏调整显示顺序，排序结果自动保存
- **🔍 角色搜索与筛选** - 团队副本、百战、试炼之地都支持按角色名、区服、账号快速搜索
- **🗡️ BOSS CD 追踪** - 25 人团队副本 BOSS 击杀状态记录与 CD 展示
- **🎁 稀有掉落统计** - 玄晶、马具、宠物、挂件、坐骑、外观、称号、秘籍等特殊掉落统计
- **⚔️ 百战/试炼之地支持** - 除团队副本外，也支持百战与试炼之地的数据管理
- **🌙 本地化存储** - 所有数据默认保存在本地设备，安装版支持应用内自动更新

## 🚀 快速上手

### 1. 下载客户端

前往 [GitHub Releases](https://github.com/lesswhite810/jx3-raid-manager/releases) 下载最新版：

- **安装版** - 支持应用内自动更新，适合长期使用
- **便携版** - 免安装，适合随身携带或手动备份

### 2. 配置游戏目录

启动应用后，在配置页填写《剑网3》游戏目录（例如 `E:\Game\SeasunGame`），程序会自动补全到正确的游戏数据目录。

### 3. 添加或扫描账号

进入"账号管理"页面后，你可以：

- 点击"**角色分析**"自动扫描游戏目录，导入本地账号与角色
- 点击"**新增账号**"手动添加账号，再在账号下添加角色

### 4. 开始记录副本数据

进入对应玩法页面后，点击"添加记录"，选择角色、副本、难度，输入工资收入即可。
