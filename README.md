# JX3 Raid Manager（剑网三副本管家）

> 一款为《剑网3》团长与多号玩家打造的本地化团队管理工具，集中解决副本收入记录、账号角色管理、掉落统计、BOSS CD 追踪等日常痛点。

## ✨ 核心功能

- **💰 智能工资结算** - 记录团队副本、百战、试炼之地等玩法收入，支持备注、明细查看与汇总统计
- **📊 数据概览** - 首页支持按时间范围查看收益、记录数、稀有掉落，团队收益和角色表现一目了然
- **👥 账号角色管理** - 支持手动添加或扫描游戏目录自动识别本地账号，支持本人/代清分类管理
- **↕️ 账号拖动排序** - 账号管理页支持直接拖动调整显示顺序，排序结果自动保存
- **🔍 角色搜索筛选** - 团队副本、百战、试炼之地都支持按角色名、区服、账号快速搜索
- **🗡️ BOSS CD 追踪** - 25 人副本 BOSS 击杀状态记录与 CD 展示，方便排班和查漏
- **🎁 稀有掉落统计** - 玄晶、马具、宠物、挂件、坐骑、外观、称号、秘籍等特殊掉落统计
- **⚔️ 百战/试炼之地** - 除团队副本外，也支持百战与试炼之地的数据管理
- **🌙 本地化存储** - 所有数据默认保存在本地设备，支持自定义数据目录
- **🔄 自动更新** - 安装版支持应用内自动更新，便携版可前往发布页手动下载

## 🚀 快速上手

### 1. 下载客户端

前往 [GitHub Releases](https://github.com/lesswhite810/jx3-raid-manager/releases) 下载最新版：

- **安装版** - 支持应用内自动更新，适合长期使用
- **便携版** - 免安装，适合随身携带或手动备份

### 2. 配置游戏目录

启动应用后，在配置页填写《剑网3》游戏目录（例如 `E:\Game\SeasunGame`），程序会自动补全到正确的游戏数据目录。

### 3. 添加或扫描账号

- **自动扫描（推荐）** - 进入"账号管理"页面，点击"角色分析"自动导入游戏目录中的本地账号与角色
- **手动添加** - 点击"新增账号"手动输入账号信息，再在账号下添加角色

### 4. 开始记录副本数据

进入对应玩法页面后，点击"添加记录"，选择角色、副本、难度，输入工资收入即可。

## ❓ 常见问题

**Q: 为什么扫描不到我的角色？**
A: 请检查配置页的游戏路径是否正确，以及是否在该电脑上登录过这些角色。

**Q: 我的数据存在哪里？**
A: 默认保存在本地设备。安装版默认在安装目录，便携版默认在用户目录，也可在配置页切换自定义数据目录。

**Q: 如何备份数据？**
A: 关闭应用后，复制数据目录下的 `jx3-raid-manager.db` 文件到安全位置即可。
