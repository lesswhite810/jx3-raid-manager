# JX3 Raid Manager (剑网三副本管家)

JX3 Raid Manager 是一个专为《剑网三》玩家打造的本地化副本数据管理工具。它帮助你记录副本收入、管理账号角色，并支持自动扫描游戏数据，让你对自己的团本收益了如指掌。

## ✨ 主要功能

- **📊 收益概览**: 实时统计金币总收入，提供趋势图和分布图，直观展示"哪天赚得多"和"哪个号最红"。
- **🗂️ 统一时间范围**: 首页概览与收益/稀有掉落详情统一支持按本周、本月、全部切换查看。
- **🛡️ 副本账本**: 详细记录每一次打本的收入、难度和时间，告别糊涂账。
- **👥 账号管家**: 统一管理多账号多角色，清晰展示角色区服信息。
- **📝 代清备注**: 支持给代清账号补充备注，并在账号列表和副本角色卡片中直接查看。
- **🔍 智能扫描**: 一键扫描游戏目录，自动导入账号和角色数据（需配置游戏路径）。
- **🔎 角色搜索**: 团队副本、百战和试炼之地都支持按角色名、区服、账号快速搜索角色卡片。
- **🎴 翻牌统计**: 试炼之地支持按角色弹出查看 1-5 号位翻牌次数、翻中装备率和位置出现率；首页会将试炼翻牌统计独立展示，并提供独立详情页查看全部历史汇总。
- **🌟 副本收藏**: 收藏常用副本，快速访问常用记录入口。
- **🌗 个性主题**: 支持深色/亮色模式切换，夜间记账不伤眼。
- **🔄 自动更新**: 安装版支持启动时自动检查更新，并可在配置页手动检查；便携版会提示前往 GitHub 下载新版本。
- **⚙️ 自动保存配置**: 配置页修改后会自动保存，无需额外点击保存按钮。

## 📖 使用说明

### 第一步：初始化配置
1. 打开应用，点击底部或侧边的 **"配置" (Settings)** 图标。
2. 在 **"游戏路径设置"** 中，点击选择你的《剑网三》安装根目录（例如 `E:\\Game\\SeasunGame`）。
   > **注意**: 运行时会自动补全到 `Game\JX3\bin\zhcn_hd`。旧配置如果已经填写到 `SeasunGame\Game\JX3\bin\zhcn_hd` 或其子目录，也会继续兼容。

### 第二步：添加账号与角色
你有两种方式添加数据：
*   **自动扫描 (推荐)**: 在 **"账号管理"** 页面，点击右上角的 **"扫描账号"** 按钮。应用会自动读取游戏本地数据，导入登录过的账号和角色。
*   **手动添加**: 在 **"账号管理"** 页面，点击 **"添加账号"**，手动输入账号名称。然后在该账号下点击 **"添加角色"**，输入角色名、服务器等信息。

### 第三步：记录副本收入
1. 打完副本分工资后，进入 **"副本管理"** 页面（或在首页点击快捷入口）。
2. 点击 **"添加记录"**。
3. 选择对应的 **角色**、**副本名称**、**难度**（如主流、普通、英雄）。
4. 输入 **工资收入**。
5. 点击保存，系统会自动更新首页的统计数据。

## ❓ 常见问题 (FAQ)

**Q: 为什么"自动扫描"扫描不到我的角色？**
A: 请检查：
1. "配置"页面的游戏路径是否设置正确。
2. 确保你已经在该电脑上登录过这些角色。
3. 请尝试手动指定到包含 `userdata` 的目录（通常是 `SeasunGame\Game\JX3\bin\zhcn_hd`）。

**Q: 我的数据存在哪里？安全吗？**
A: 数据存储在你的本地电脑中。
- **存储路径**: `C:\Users\你的用户名\.jx3-raid-manager\` (在用户主目录下)。
- **数据文件**: `jx3-raid-manager.db` (这是一个 SQLite 数据库文件)。
- **安全性**: 数据完全本地化，不经过任何云端服务器，隐私绝对安全。

**Q: 如何备份和恢复数据？**
A:
- **备份**: 关闭应用，将 `C:\Users\你的用户名\.jx3-raid-manager\jx3-raid-manager.db` 文件复制到安全的地方。
- **恢复**: 关闭应用，将备份的 `.db` 文件覆盖回原目录即可。

**Q: 首页的图表为什么是空的？**
A: 图表需要基于你添加的副本记录生成。当你开始添加第一条副本收入记录后，图表就会自动出现。
