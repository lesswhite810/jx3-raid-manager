# 副本系统性调整与优化计划

## 1. 副本难度模式配置更新

### 1.1 为《剑胆琴心》《风骨霸刀》《重制版》及《世外蓬莱》资料片的所有副本统一新增25人普通难度模式

* 为每个副本添加25人普通模式，默认设置为禁用状态

* 需要修改的副本：

  * 剑胆琴心：永王行宫·仙侣庭园、永王行宫·花月别院

  * 风骨霸刀：上阳宫·观风殿、上阳宫·双曜亭、风雷刀谷·锻刀厅、风雷刀谷·千雷殿

  * 重制版：狼牙堡·战兽山、狼牙堡·燕然峰、狼牙堡·辉天堑、狼神殿

  * 世外蓬莱：荒血路、青莲狱、巨冥湾、饕餮洞、敖龙岛、范阳夜变

### 1.2 永久移除上述所有副本的10人挑战难度模式

* 删除所有10人挑战模式的副本条目

* 涉及副本：

  * 剑胆琴心：永王行宫·仙侣庭园（10人挑战）、永王行宫·花月别院（10人挑战）

  * 风骨霸刀：无10人挑战模式

  * 重制版：无10人挑战模式

  * 世外蓬莱：荒血路（10人挑战）、青莲狱（10人挑战）、巨冥湾（10人挑战）、饕餮洞（10人挑战）、敖龙岛（10人挑战）、范阳夜变（10人挑战）

## 2. 副本界面显示优化

### 2.1 移除副本禁用后的冗余视觉元素

* 删除副本卡片右上角的EyeOff图标

* 删除副本名称下方的"已禁用"文本描述

* 删除副本卡片底部的"此副本已禁用"文本描述

### 2.2 删除副本难度禁用后的橘色点状标识

* 删除难度选择区域左上角的橘色点状标识（默认禁用标识）

## 3. 副本记录排序规则调整

### 3.1 实现按人数规模区分的排序规则

* 25人难度副本优先显示

* 10人难度副本随后显示

### 3.2 实现按资料片版本倒序排列

* 在相同人数规模内，按照资料片版本倒序排列

* 最新版本资料片的副本优先显示

## 4. 跨系统一致性保障

### 4.1 确保所有相关系统中完全同步生效

* 团队查找器、副本入口界面、难度选择界面、副本记录系统及相关API接口

* 保证改动仅影响指定资料片的副本，不对其他未指定资料片的副本难度配置产生任何影响

### 4.2 实施与验证要求

* 完成所有配置变更后，进行全面的功能测试

* 验证新增难度的可用性、移除难度的彻底性及排序规则的准确性

* 进行跨系统一致性测试，确保所有相关界面和功能模块显示一致、数据同步

## 5. 实施步骤

1. 修改 `staticRaids.ts` 文件，添加25人普通模式并删除10人挑战模式
2. 修改 `RaidManager.tsx` 组件，删除冗余的视觉元素
3. 运行构建测试，确保代码编译通过
4. 进行功能测试，验证副本显示和交互正常
5. 进行回归测试，确认改动未对其他系统产生意外影响

此计划将全面优化副本系统的配置和显示，提升用户体验并确保系统稳定性。
